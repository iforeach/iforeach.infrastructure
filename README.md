#### 开发说明
1. 本项目的代码为参考第三方代码进行修订，其中添加了部分自我意愿，因此代码的名称空间不再使用原有的定义，全部修订为 org.iForeach。
2. 所有的代码都会通过 README.md 进行说明引用与参考的源信息，也会在公开发行在nuget上的类库包含同样的描述，如有疑议请与本人联系改进。
3. 本项目系统属于糅合项目，因此所参考、借鉴的第三方项目将不仅仅局限于开源项目。
